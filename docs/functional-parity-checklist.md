# Functional Parity Checklist

Цель: зафиксировать текущий статус покрытия функциональности по ключевым manager/admin экранам и целевое состояние.

## Матрица as-is vs to-be

| Модуль | Сценарий | Роль | As-is | To-be |
|---|---|---|---|---|
| Dashboard | Просмотр агрегатов ESSI/рисков/продуктивности | manager/admin | Есть | Есть + явный выбор периода и расширенный drill-down |
| Dashboard | Переходы из виджетов в профильные страницы | manager/admin | Частично | Полное покрытие для всех KPI-виджетов |
| Dashboard | Состояния пустых данных/ошибок по секциям | manager/admin | Частично | Явные fallback-стейты по секциям |
| Departments | Карточка отдела с агрегатами и списком сотрудников | manager/admin | Частично | Полный контекст отдела с риском и переходами |
| Employees | Поиск и фильтрация | manager/admin | Клиентская фильтрация | Серверная фильтрация с пагинацией |
| Employees | Пагинация списка | manager/admin | Нет | Серверная пагинация и навигация страниц |
| Departments | Поиск/сортировка/пагинация списка | manager/admin | Нет/частично | Серверные параметры + UI контролы |
| Reports | Создание отчётов PDF/XLSX | manager/admin | Есть | Есть |
| Reports | История экспортов и статусы | manager/admin | Частично | Отдельный блок истории и retry failed |
| Reports | Мониторинг импортов (jobs) | manager/admin | Частично | История импортов с актуальными статусами |
| Recommendations | Просмотр и выполнение рекомендаций | manager/admin | Есть | Есть + фильтры и поиск |
| Settings/Admin | Создание пользователей | admin | Есть | Есть |
| Settings/Admin | Список/редактирование пользователей | admin | Нет | CRUD-lite (редактирование роли/привязки) |
| Settings/Admin | Сброс пароля пользователя | admin | Нет | Есть |
| Settings/Admin | Аудит с фильтрами/экспортом | admin | Частично | Фильтры по периоду + CSV экспорт |

## Definition of Done

- Для каждого сценария выше подтверждено поведение в UI и API.
- Для manager/admin проверены ограничения доступа и ошибки бизнес-правил.
- Критичные потоки не регрессировали: login, dashboard, employees, departments, reports, recommendations, settings/admin.
