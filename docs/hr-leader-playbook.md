# HR and Leader Playbook

## Цель сценария

Показать цикл управления устойчивостью персонала:
1) обнаружить проблему,
2) внедрить меры,
3) зафиксировать событие,
4) проверить эффект повторным замером.

## Сценарий

### Шаг 1. Диагностика

- HR открывает дашборд и смотрит:
  - общий ESSI,
  - 5 частных индексов по блокам,
  - динамику за период.
- Если есть низкие блоки (ниже 60), фиксируется гипотеза причины.

### Шаг 2. Локализация

- Переход в проблемный отдел.
- Анализ «кто формирует риск/падение».
- Переход в карточки сотрудников для просмотра блоковых отклонений.

### Шаг 3. Управленческие меры

- Руководитель и HR выбирают меры:
  - обучение,
  - изменение KPI,
  - процессные изменения,
  - коммуникационные сессии.
- Мера фиксируется как управленческое событие в разделе «Отчеты».

### Шаг 4. Повторный замер

- Через 4–8 недель проводится повторный опрос.
- На графике динамики сравниваются значения до/после события.

### Шаг 5. Оценка эффекта

- Оценивается дельта:
  - общий ESSI,
  - ключевые блоки,
  - доля сотрудников в зоне риска.
- При недостаточном эффекте запускается новый цикл корректировки.

## Рекомендованные пороги

- ESSI >= 80: поддерживать текущие практики.
- ESSI 60–79: точечные улучшения и регулярный мониторинг.
- ESSI 40–59: приоритетный план действий с ответственными.
- ESSI < 40: срочные меры и короткий цикл повторного контроля.
